## ❓ Frequently Asked Questions

### Is Superior Drummer suitable for beginners?

Yes. While it offers advanced features for professionals, beginners can quickly start using the included presets, drum kits, and MIDI grooves.

### Which DAWs are supported?

Superior Drummer supports most major Digital Audio Workstations that accept VST3, AU, or AAX plugins.

### Can I use third-party MIDI files?

Yes. You can import standard MIDI drum files and customize them using Superior Drummer's editing tools.

### Does it support expansion libraries?

Yes. Superior Drummer is compatible with Toontrack's SDX and EZX expansion libraries, allowing you to expand your drum collection.

### Is it suitable for commercial music production?

Yes. Projects created using a properly licensed version can generally be used for commercial releases in accordance with Toontrack's license agreement.
